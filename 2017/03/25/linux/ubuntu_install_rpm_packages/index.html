rpm 和 deb

  • rpm 包
    rpm 后缀的安装包，是可执行文件安装包，可以用 rpm 命令进行安装。RPM，RedHat Package Manager。

  • deb 包
    deb 后缀的安装包，debian 及其衍生版本的安装包（如 ubuntu），是可执行文件安装包。可以用 dpkg 命令进行安装。
