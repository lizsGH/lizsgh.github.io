Linux 终端窗口快捷键

  • Ctrl + p: 上一条命令 (相当于向上箭头)。

  • Ctrl + n: 下一条命令 (相当于向下箭头)。

  • Ctrl + a: 光标回到命令行开始位置。

  • Ctrl + e: 光标回到命令行开始位置。

  • Ctrl + b: 光标向左移动一位。

  • Ctrl + f: 光标向右移动一位。

  • Ctrl + h: 删除光标左边的一个字符。

  • Ctrl + d: 删除光标右边的一个字符。

  • Ctrl + w: 删除光标左边的一个’单词’字符。

  • Ctrl + u: 删除光标左边的所有字符。

  • Ctrl + k: 删除光标右边的所有字符。

  • Ctrl + l: 清屏。
